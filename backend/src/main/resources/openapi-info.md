# API Entrypoint

## Base URL

`/api/v1`

## OpenAPI UI

- Swagger UI (default): `/swagger-ui/index.html`
- OpenAPI JSON (default): `/v3/api-docs`

## Initial endpoint groups

- Health: `/api/v1/health`
- Readiness: `/api/v1/ready`
- Users: `/api/v1/users`
- Products: `/api/v1/products`
- Orders: `/api/v1/orders`
- Order items: `/api/v1/order-items`
- Auth: `/api/v1/auth/login`
- Search: `/api/v1/search`, `/api/v1/search/advanced`
